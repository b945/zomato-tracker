**Food Stats** is a Chrome extension that gives you deep insights into your Zomato ordering habits. It analyzes your order history to provide a beautiful, privacy-first dashboard showing your total spending, top restaurants, and ordering trends.

## ğŸš€ Installation (Local Development)

If you want to run this extension locally or contribute to the code:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Jobin-S/Food-stats-extenstion.git
   cd Food-stats-extenstion
   ```

2. **Load into Chrome**
   - Open Chrome and navigate to `chrome://extensions/`.
   - Toggle **Developer mode** in the top-right corner.
   - Click **Load unpacked**.
   - Select the project directory you just cloned.

3. **Pin the Extension**
   - Click the puzzle piece icon in your Chrome toolbar.
   - Pin **Food Stats** for easy access.

## ğŸ“– How to Use

1. **Log in to Zomato**: Ensure you are logged into your account on [zomato.com](https://www.zomato.com).
2. **Open the Extension**: Click the **Food Stats** icon in your toolbar.
3. **Sync Data**: Click the **"Sync Data"** button. The extension will open Zomato (if not already open) and safely fetch your order history.
4. **Explore Insights**: Once the sync is complete, the dashboard will populate with your personalized stats and charts.
